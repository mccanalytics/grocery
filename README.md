# Grocery List

A simple, web-based grocery list application for organizing your shopping needs.

## Files

- **grocery-list.html** - Main grocery list application
- **grocery-list-guide.html** - User guide and instructions
- **thoughtful-files-icon.png** - Application icon

## Usage

Open `grocery-list.html` in your web browser to start using the grocery list application.

For detailed instructions, see `grocery-list-guide.html`.
